## 🛠 快速开始

```bash
# 安装依赖
npm install

# 本地预览
hexo server

# 新建文章
hexo new "文章标题"

# 清理并构建
hexo clean && hexo generate

# 部署到 GitHub Pages
hexo deploy
```
